# 开发指南

## 环境设置

### 安装 uv

```bash
# macOS/Linux
curl -LsSf https://astral.sh/uv/install.sh | sh

# Windows
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```

### 克隆并安装

```bash
git clone https://github.com/afine907/nano-code.git
cd nano-code
uv sync
```

### 配置环境变量

```bash
cp .env.example .env
# 编辑 .env 文件，填入 API Key
```

## 常用命令

### 运行 CLI

```bash
uv run nano-code
```

### 运行测试

```bash
# 运行所有测试
uv run pytest tests/ -v

# 运行特定测试
uv run pytest tests/test_agent.py -v

# 带覆盖率
uv run pytest tests/ -v --cov=src/nano_code --cov-report=html
```

### 代码质量

```bash
# Linting
uv run ruff check src/ tests/

# 格式化
uv run ruff format src/ tests/

# 类型检查
uv run mypy src/
```

### 构建发布

```bash
# 构建包
uv build

# 发布到 PyPI
uv publish
```

## 代码风格

- **行长度**：100 字符
- **Python 版本**：3.11+
- **类型注解**：必须使用
- **格式化工具**：Ruff
- **类型检查**：mypy strict

### 示例

```python
def execute_node(state: AgentState) -> dict[str, Any]:
    """执行节点：运行工具调用

    Args:
        state: 当前 Agent 状态

    Returns:
        更新后的状态片段
    """
    registry = get_tool_registry()
    results: list[str] = []

    for tool_call in state["tool_calls"]:
        result = registry.execute(tool_call["name"], tool_call["args"])
        results.append(result)

    return {"tool_results": results, "tool_calls": []}
```

## 测试标记

```python
@pytest.mark.e2e      # 真实 API 调用
@pytest.mark.longcat  # LongCat API 测试
@pytest.mark.slow     # 慢测试
```

运行特定标记：

```bash
uv run pytest -m "not e2e" tests/  # 跳过 E2E 测试
```

## 添加新功能

### 1. 添加新工具

```python
# tools/my_tool.py
from langchain_core.tools import tool

@tool
def my_tool(input: str) -> str:
    """工具描述"""
    return "result"
```

```python
# tools/registry.py
from nano_code.tools.my_tool import my_tool

default_tools = [..., my_tool]
```

```python

def my_node(state: AgentState) -> dict:
```
