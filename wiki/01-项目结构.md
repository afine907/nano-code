# Nano-Code 项目结构

## 目录概览

```
nano-code/
├── src/nano_code/          # 源代码
│   ├── agent/              # Agent 核心（LangGraph 状态机）
│   │   ├── graph.py        # 状态图定义
│   │   ├── nodes.py        # 节点实现
│   │   └── state.py        # 状态定义
│   ├── cli/                # 命令行界面
│   │   ├── main.py         # CLI 主入口
│   │   ├── console.py      # 控制台输出
│   │   └── enhanced.py     # 增强功能
│   ├── core/               # 核心模块
│   │   ├── llm.py          # LLM 客户端
│   │   ├── config.py       # 配置管理
│   │   └── ...             # 其他核心功能
│   ├── memory/             # 记忆管理
│   │   └── conversation.py # 对话记忆
│   └── tools/              # 工具系统
│       ├── registry.py     # 工具注册中心
│       ├── file_tools.py   # 文件操作
│       ├── shell_tools.py  # Shell 命令
│       └── search_tools.py # 搜索工具
├── tests/                  # 测试代码
├── wiki/                   # 项目文档
├── pyproject.toml          # 项目配置
└── README.md               # 项目说明
```

## 核心文件说明

### Agent 层

| 文件 | 职责 |
|------|------|
| [agent/graph.py](../src/nano_code/agent/graph.py) | LangGraph 状态图定义，构建 Agent 循环 |
| [agent/nodes.py](../src/nano_code/agent/nodes.py) | thinking_node 和 execute_node 实现 |
| [agent/state.py](../src/nano_code/agent/state.py) | AgentState TypedDict 状态定义 |

### CLI 层

| 文件 | 职责 |
|------|------|
| [cli/main.py](../src/nano_code/cli/main.py) | CLI 主入口，交互式会话 |
| [cli/console.py](../src/nano_code/cli/console.py) | Rich 控制台输出、状态栏、动画 |
| [cli/enhanced.py](../src/nano_code/cli/enhanced.py) | 增强交互功能 |

### Core 层

| 文件 | 职责 |
|------|------|
| [core/llm.py](../src/nano_code/core/llm.py) | LLM 客户端（OpenAI/Anthropic/兼容 API） |
| [core/config.py](../src/nano_code/core/config.py) | Pydantic Settings 配置管理 |

### Memory 层

| 文件 | 职责 |
|------|------|
| [memory/conversation.py](../src/nano_code/memory/conversation.py) | 对话记忆、Token 计数、自动压缩 |

### Tools 层

| 文件 | 职责 |
|------|------|
| [tools/registry.py](../src/nano_code/tools/registry.py) | 工具注册、执行、LangChain 工具转换 |
| [tools/file_tools.py](../src/nano_code/tools/file_tools.py) | read_file, write_file, edit_file, list_directory |
| [tools/shell_tools.py](../src/nano_code/tools/shell_tools.py) | run_command 执行 Shell 命令 |
| [tools/search_tools.py](../src/nano_code/tools/search_tools.py) | grep_search, glob_search 搜索工具 |

## 架构层次

```
┌─────────────────────────────────────────┐
│            CLI Layer (cli/)             │
│  用户交互、命令处理、输出渲染              │
└─────────────────┬───────────────────────┘
                  │
┌─────────────────▼───────────────────────┐
│          Agent Loop (agent/)            │
│  LangGraph 状态机: thinking → execute   │
└─────────────────┬───────────────────────┘
                  │
┌─────────────────▼───────────────────────┐
│          Tool Layer (tools/)            │
│  文件操作、Shell 命令、搜索工具           │
└─────────────────────────────────────────┘
```
