# 记忆系统

## 概述

记忆系统管理对话历史，支持 Token 计数和自动压缩。

## ConversationMemory 类

### 初始化

```python
memory = ConversationMemory(
    max_tokens=100000,      # 最大 token 数
    storage_path=Path("~/.nano-code/sessions/default.json"),
    auto_save=True          # 自动保存
)
```

### 核心方法

| 方法 | 功能 |
|------|------|
| `add_message(message)` | 添加消息到历史 |
| `token_count()` | 计算当前 token 数 |
| `get_context()` | 获取所有消息 |
| `get_last_n_messages(n)` | 获取最近 N 条消息 |
| `clear()` | 清空记忆 |
| `save()` | 保存到文件 |
| `load()` | 从文件加载 |

## Token 计数

使用 `tiktoken` 库计算 token：

```python
import tiktoken
encoding = tiktoken.encoding_for_model("gpt-4")

def token_count(self) -> int:
    total = 0
    for msg in self.messages:
        total += len(self._encoding.encode(msg.content))
    return total
```

## 自动压缩

当 token 超过 `max_tokens` 时触发压缩：

```python
def _compress(self) -> None:
    # 保留系统消息
    system_messages = [m for m in self.messages if isinstance(m, SystemMessage)]

    # 保留最近 20 条普通消息
    recent_messages = other_messages[-20:]

    # 添加压缩摘要
```

## 持久化存储

### 保存格式

```json
{
  "messages": [
    {"type": "HumanMessage", "content": "用户输入"},
    {"type": "AIMessage", "content": "AI 回复"}
  ]
}
```

### 存储路径

默认存储在 `~/.nano-code/sessions/` 目录：

```
~/.nano-code/
├── sessions/
│   └── default.json    # 默认会话
└── history.txt         # 命令历史
```

## 使用示例

### CLI 中使用

```python
# 初始化记忆
memory = ConversationMemory(
    storage_path=settings.storage_path / "sessions" / "default.json",
    auto_save=True
)

# 添加用户消息
memory.add_message(HumanMessage(content=user_input))

# 运行 Agent
result = graph.invoke(state)

# 添加 AI 回复
memory.add_message(AIMessage(content=response))
```

### 手动管理

```python
# 获取上下文
context = memory.get_context()

# 获取最近 5 条
recent = memory.get_last_n_messages(5)

# 清空记忆
memory.clear()
```
