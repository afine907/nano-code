# 09-PR交互增强

> CLI 交互体验增强功能的变更说明

## 📋 变更概述

为 Nano-Code 添加丰富的交互增强功能，提升用户体验和可用性。

## ✨ 新增功能

### 1. 会话统计功能
- **消息计数**: 实时统计用户与助手的消息数量
- **工具调用统计**: 记录本次会话中的工具调用次数
- **Token 统计**: 显示当前会话的 token 使用量
- **会话时长**: 显示从会话开始到现在的时长

### 2. 状态栏
每次交互后显示状态栏，包含：
- 当前模型名称
- 消息数 / 工具调用数
- 会话时长
- Token 使用量

### 3. 新增命令

| 命令 | 功能 |
|------|------|
| `/stats` | 显示详细的会话统计信息 |
| `/model` | 显示当前使用的模型 |
| `/history` | 显示最近的消息历史 |
| `/reset-stats` | 重置会话统计 |

### 4. 交互增强
- **思考动画**: 处理请求时显示旋转动画，提升用户体验
- **历史建议**: 输入时自动提示历史命令
- **增强的帮助信息**: 更清晰的命令说明和使用技巧
- **工具调用可视化**: 显示工具调用序号和参数

### 5. 输出格式优化
- 工具结果支持时长显示
- 长结果自动截断并显示字符数
- 更清晰的图标和颜色区分

## 📁 变更文件

- `src/nano_code/cli/console.py` - 增强控制台输出功能
